# 30 React Projects 


## How to Run Any Project

Each `.jsx` file is a self-contained React component with a default export.

### Quick Start (Vite)
```bash
npm create vite@latest my-app -- --template react
cd my-app
npm install
# Copy any .jsx file into src/App.jsx (replace contents)
npm run dev
```

### Or use StackBlitz / CodeSandbox
Paste the file contents directly — they run instantly.

## Key Concepts Covered

- **Hooks**: useState, useEffect, useRef, useReducer, useCallback, useMemo, useContext
- **Custom Hooks**: useDebounce, useLocalStorage
- **Performance**: Debouncing, memoization, IntersectionObserver
- **APIs**: Web Audio API, Canvas API, Drag & Drop API, Clipboard API
- **Patterns**: Compound components, Context + Provider, Reducer pattern
- **Algorithms**: SM-2 spaced repetition, flood fill, Morse encoding
